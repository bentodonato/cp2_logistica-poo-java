Sistema de Logística com POO em Java para a CP2



Projeto desenvolvido para a CP2 de Programação Orientada a Objetos (POO) em Java.





\- Organização em pacotes (`model` e `app`)





Estrutura do Projeto



\- `model` → classes de domínio (Veiculo, Caminhao, Pacote, etc.)

\- `app` → classe principal (`Main`)





Mudanças que ocorreram do Astah para o código:

- StatusPacote se tornou StatusPack, e se tornou um enum, e não mais um diagrama de classes
- Veiculo é Abstract







Para rodar o projeto, execute a classe:



`app.Main`
